# 🗺️ Spatial Statistics Project

## Overview

This repository contains projects completed for the **Spatial Statistics** course. It includes R scripts, R Markdown files, research papers, and HTML reports generated from R Markdown. The projects demonstrate the application of spatial statistical methods for analyzing geographic and spatially referenced data.

## Repository Contents

- 📂 **code/** – R scripts (`.R`) and R Markdown (`.Rmd`) used for spatial data analysis.
- 📂 **paper/** – Research papers and project reports in PDF format.
- 📂 **html/** – HTML reports generated from R Markdown.

## Software & Tools

- R
- RStudio
- R Markdown

## Purpose

The purpose of this repository is to document academic coursework and demonstrate the application of spatial statistical techniques for analyzing, modeling, and interpreting spatial data using R.
